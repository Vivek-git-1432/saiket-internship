API Documentation

Complete guide to the User REST API endpoints

Home Dashboard
Base URL
All API endpoints are prefixed with: http://localhost:5000/api/users
GET /api/users

Retrieve all users from the database.

Response Example 
{
  "users": [
    {
      "id": 1,
      "name": "Aarav Sharma",
      "email": "aarav@gmail.com",
      "age": 25
    },
    {
      "id": 2,
      "name": "Priya Mehta",
      "email": "priya@gmail.com",
      "age": 22
    }
  ]
}
GET /api/users/:id

Retrieve a specific user by their ID.

Response Example 
{
  "id": 1,
  "name": "Aarav Sharma",
  "email": "aarav@gmail.com",
  "age": 25
}
Error Response (404) 
{
  "success": false,
  "message": "User not found"
}
POST /api/users

Create a new user. All fields are required.

Request Body 
{
  "name": "John Doe",
  "email": "john@example.com",
  "age": 30
}
Success Response (201) 
{
  "success": true,
  "user": {
    "id": 4,
    "name": "John Doe",
    "email": "john@example.com",
    "age": 30
  }
}
Error Response (400) 
{
  "success": false,
  "message": "All fields are required"
}
PUT /api/users/:id

Update an existing user. You can update any or all fields.

Request Body 
{
  "name": "John Updated",
  "email": "john.updated@example.com",
  "age": 31
}
Success Response (200) 
{
  "success": true,
  "user": {
    "id": 1,
    "name": "John Updated",
    "email": "john.updated@example.com",
    "age": 31
  }
}
DELETE /api/users/:id

Delete a user by their ID.

Success Response (200) 
{
  "success": true,
  "message": "User deleted successfully"
}
Testing with Postman
Steps to test:
  1. Open Postman application
  2. Set the request method (GET, POST, PUT, DELETE)
  3. Enter the endpoint URL: http://localhost:5000/api/users
  4. For POST/PUT requests, go to Body → raw → JSON and add the request body
  5. Click Send and view the response
